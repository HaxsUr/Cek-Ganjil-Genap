
# Aplikasi Cek Ganjil Genap + Prima

Dengan Aplikasi Ini kita dapat Mencek ganjil genapnya angka kita inputkan.

## Fitur Aplikasi

- Mencek angka ganjil
- Mencek angka genap
- Mencek angka prima


## Keunggulan Aplikasi
- Dapat mencek data prima, genap, dan ganjil dengan tepat dan cepat
